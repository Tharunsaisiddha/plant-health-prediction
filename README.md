# 🌿 Plant Health Prediction Using CNN

## 📌 Project Overview

Plant Health Prediction Using CNN is an AI-powered deep learning application designed to analyze plant leaf images and predict their health condition. The system uses a Convolutional Neural Network (CNN) to identify visual patterns in plant leaves and classify the uploaded image into one of three categories:

- 🌱 Healthy
- 🍂 Powdery
- 🦠 Rust

The trained CNN model is integrated with an interactive Streamlit web application, allowing users to upload a plant leaf image and receive an instant prediction along with the model's confidence score.

## 🎯 Objective

The main objective of this project is to demonstrate the practical application of Artificial Intelligence, Deep Learning, and Computer Vision in plant health monitoring. The system aims to provide a simple and accessible method for identifying common plant leaf conditions through image-based classification.

## ✨ Key Features

- Upload plant leaf images in JPG, JPEG, or PNG format
- CNN-based automated leaf image classification
- Classification into Healthy, Powdery, and Rust categories
- Displays prediction results with confidence scores
- Simple and user-friendly Streamlit interface
- Provides plant health information and basic recommendations
- Accessible through a deployed web application

## 🧠 Model Architecture

The project uses a Convolutional Neural Network (CNN), a deep learning architecture particularly effective for image classification tasks. The CNN automatically learns important visual features from plant leaf images during training.

The model processes the input image through convolutional and pooling operations to extract meaningful features. These learned features are then used by the classification layers to predict the appropriate plant health category.

## 🚀 How the Application Works

1. The user opens the Streamlit web application.
2. A plant leaf image is uploaded through the application interface.
3. The uploaded image is resized and preprocessed according to the CNN model's input requirements.
4. The trained CNN model analyzes the visual features of the leaf.
5. The model predicts one of three classes: Healthy, Powdery, or Rust.
6. The predicted plant health condition and confidence score are displayed to the user.
7. Relevant plant health information and recommendations are presented based on the prediction.

## 📊 Model Performance

The CNN model achieved approximately **94% test accuracy**, demonstrating strong performance in classifying plant leaf images across the three supported categories.

## 🌐 Deployment

The application is deployed using Streamlit Community Cloud. The source code and trained CNN model are maintained in this GitHub repository, allowing the deployed application to run independently without requiring Google Colab or a temporary Cloudflare tunnel.
